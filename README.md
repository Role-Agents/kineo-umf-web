# Kineo-UMF Web

Centro Médico de Rehabilitación Física Especializada — Next.js 14 website.

## Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **Deployment**: Vercel

## Getting Started
```bash
npm install
npm run dev
```
